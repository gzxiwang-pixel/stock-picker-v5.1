# 评分制选股策略 V5.1

## 📊 策略简介

基于多指标评分系统的A股选股策略，采用量化评分方式筛选优质股票。

### 核心特性

- **评分制度**：满分100分，50分以上入选
- **多维指标**：7个技术指标综合评分
- **自动推送**：通过PushPlus实时推送选股结果
- **可视化展示**：生成精美的HTML展示页面

## 🎯 评分标准

| 指标 | 条件 | 得分 |
|------|------|------|
| 均线多头 | MA5 > MA20 | +20分 |
| BOLL开口 | 价格 > BOLL上轨 × 0.95 | +15分 |
| MACD强势 | DIF > 0 | +15分 |
| 涨幅条件 | 2% ≤ 涨幅 ≤ 8% | +15分 |
| 量能放大 | 量比 > 1.5 | +10分 |
| 成交活跃 | 成交额 > 1亿 | +10分 |
| MACD金叉 | MACD柱 > 0 | +15分 |

**入选门槛**：总分 ≥ 50分

## 📊 策略逻辑

### 数据来源

- 股票列表：腾讯财经实时行情接口
- K线数据：腾讯财经前复权K线接口
- 实时行情：腾讯财经实时行情接口

### 计算流程

1. 获取所有A股列表（过滤ST、退市股）
2. 逐只获取实时行情数据
3. 获取日K线数据计算技术指标
4. 根据7个维度计算评分
5. 筛选总分≥50分的股票
6. 按得分降序排序
7. 生成HTML展示页面
8. 保存JSON数据文件
9. 发送PushPlus推送

### 技术指标说明

#### 1. 均线多头（20分）
- **计算**：MA5 > MA20
- **意义**：短期均线上穿中期均线，趋势向好

#### 2. BOLL开口（15分）
- **计算**：当前价格 > BOLL上轨 × 0.95
- **意义**：股价突破布林带上轨，强势特征

#### 3. MACD强势（15分）
- **计算**：DIF > 0
- **意义**：MACD在零轴上方，多头强势

#### 4. 涨幅条件（15分）
- **计算**：2% ≤ 当日涨幅 ≤ 8%
- **意义**：涨幅适中，避免追高和弱势股

#### 5. 量能放大（10分）
- **计算**：量比 > 1.5
- **意义**：成交量放大，资金关注度高

#### 6. 成交活跃（10分）
- **计算**：成交额 > 1亿元
- **意义**：流动性好，资金充裕

#### 7. MACD金叉（15分）
- **计算**：MACD柱 > 0（红柱）
- **意义**：DIF在DEA上方，买入信号
